# Tauri Toolkit
> Download MySQL data, data pivot...

![image](https://github.com/tansen87/tauri-toolkit/assets/98570790/bb98f299-da9b-4616-9ef5-28765997bf49)

## Usage
### Installation dependencies
```bash
npm i -g pnpm
pnpm i
```
### run web
```bash
pnpm dev
```
### run desktop
```bash
pnpm run tauri dev
```
### build
```bash
pnpm run tauri build
```
