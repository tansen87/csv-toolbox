# Tauri Toolkit
> Download MySQL data, data pivot...
![image](https://github.com/tansen87/tauri-toolkit/assets/98570790/e616a119-857e-46d5-9973-1b4c141084c4)

## Usage
### Installation dependencies
```bash
npm i -g pnpm
pnpm i
```
### run web
```bash
pnpm dev
```
### run desktop
```bash
pnpm run tauri dev
```
### build
```bash
pnpm run tauri build
```
